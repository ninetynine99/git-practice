# git practice (2018-02-10 스터디 정리)

### 1. 리눅스 명령어
#### print working directory
    pwd
#### pwd에 있는 폴더 확인
    ls
#### change directory
    cd
* cd 폴더명 -> 해당 폴더로 이동
* cd .. -> 상위폴더로 이동
#### make directory
  mkdir 만들 폴더명
